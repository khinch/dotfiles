# dotfiles
My dotfiles for linux tools

# Usage
Use `stow` to create symlinks to dotfiles to quickly recreate a familar and productive dev environment

# Tools
## Alacritty
`stow alacritty`
## Neovim
`stow neovim`
## Starship
```
stow starship 
cat <<EOF >> ~/.bashrc
eval "\$(starship init bash)"
EOF
```
## Yazi
`stow yazi`
